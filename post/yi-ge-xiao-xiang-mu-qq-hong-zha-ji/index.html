一个小项目：qq轰炸机

2021-11-05 计算机

前几天由于需要和许多人进行聊天，所以萌生了写个群发器的想法，但是在学习群发器的过程中，顺手做了个轰炸机。

CODE：

#include<stdio.h>
#include<windows.h>

int main()
{

char name[30] = { 0 };
printf("选择轰炸对象\n");
scanf_s("%s", name,30);
int times = 0;
printf("次数\n");
scanf_s("%d", &times);
HWND qqhwnd = FindWindowA(NULL, name);
for (int i = 0;i < times; i++) 
{
	SendMessageA(qqhwnd, WM_PASTE, 0, 0);
	SendMessageA(qqhwnd, WM_KEYDOWN, VK_RETURN, 0);
}
return 0;

}
